# Shonan Code Summit（ShoCode）行動規範

## はじめに

Shonan Code Summit（ShoCode）は、エンジニア、開発者、テクノロジー愛好家が集まり、
アイデアや知識、経験を共有するコミュニティ主導のイベントです。
すべての参加者にとって安全で、インクルーシブで、互いを尊重する環境を築くため、
この行動規範を定めています。

この行動規範は、イベント本体をはじめ、オンラインでの議論、ソーシャルメディアでのやり取り、
その他コミュニティ活動を含む、ShoCode に関連するすべての活動に適用されます。
これらの活動に参加することで、この行動規範に同意し、その原則を守ることに同意したものとみなされます。

## 私たちの価値観

多様性、インクルージョン、そして相互尊重は、活気ある、生産的で、
歓迎される​コミュニティを築くために不可欠だと考えています。
私たちのコミュニティは、年齢、人種、民族、国籍、宗教、性別、性的指向、能力、
経歴を問わず、すべての人を歓迎します。
オープンさ、好奇心、そしてコラボレーションを大切にし、
誰もが安心して考えやアイデア、フィードバックを共有できる環境を維持することに努めます。

## 期待される行動

ShoCode の参加者は全員、バックグラウンドや立場にかかわらず、
他の参加者に対してプロフェッショナルで、礼儀正しく、敬意ある態度で接することが求められます。
具体的には、以下のことを期待します。

- 差別的、攻撃的、またはハラスメントにあたる言動（侮辱、誹謗中傷、脅迫、暴力を含む）を行わない。
- 他者の境界線、プライバシー、パーソナルスペースを尊重し、明確な同意なしに身体的接触をしない。
- 自分の言葉や行動が周囲に与える影響を意識し、個人やグループに対する決めつけや一般化を避ける。
- 相手の話に積極的かつ共感を持って耳を傾け、遮ったり話しかぶせたりしない。
- 自分自身の行動に責任を持ち、他者を傷つけたり不快にさせた場合には謝罪する。
- ハラスメントや差別にあたる行為を見たり体験した場合は、直ちにイベントの主催者またはモデレーターに報告する。

## 許容されない行為

以下の行為は、ShoCode に関連するすべての活動において許容されず、一切容認されません。

- 年齢、人種、民族、国籍、宗教、性別、性的指向、能力、法律や社会規範で
  保護されるその他の特性に基づく差別的な言動。
- 参加者や集団に向けた攻撃的な言動（侮辱、誹謗中傷、脅迫、暴力を含む）。
- ハラスメントやいじめ（望まない身体的接触、つきまとい、威圧、強制を含む）。
- 大声を出す、騒ぐ、イベントのルールやガイドラインを無視するなど、
  他の参加者のイベント参加や体験を妨げる破壊的・不敬な行為。

## 許容されない行為への対応

許容されない行為を行った参加者は、警告なくイベントやコミュニティ活動からの
即時退場を求められることがあります。
さらに、主催者やモデレーターは、個人への警告、今後の活動への参加制限、
適切な当局への報告など、状況に応じた措置を講じることがあります。

## 報告と適用

この行動規範に反する行為を経験したり目撃した場合、
または他の参加者の行為について疑問や懸念がある場合は、
直ちにイベントの主催者またはモデレーターに報告してください。
すべての報告を真摯に受け止め、迅速かつ秘密厳守で調査します。

**報告先:** Drew Robbins — drew@drewby.com

## おわりに

Shonan Code Summit（ShoCode）は、すべての参加者にとって多様で、インクルーシブで、
互いを尊重するコミュニティを築くことに尽力しています。
コミュニティメンバー全員の貢献を大切にし、この行動規範を守っていただくことに感謝いたします。
