[![English](https://img.shields.io/badge/🇬🇧-English-white?style=plastic)](README.md)

> ビルダー、クリエイター、実行する人たちのための、コミュニティ主導の完全バイリンガル（日本語 / English）
ワンデイイベント。エージェンティック・コーディングの最前線を探求します。
神奈川県湘南エリアのインスピレーショナルな会場で開催。

このリポジトリは、イベントの詳細、参加登録、コミュニティへの貢献のための唯一の情報源です。
ShoCode の原点に忠実に、イベントのあらゆる面を GitHub で管理します。
参加登録も Pull Request を通じて行います（PR に慣れていない方向けの簡単な方法もあります）。

## イベント概要

Shonan Code Summit（ShoCode）は、**エージェンティック・コーディング**をテーマに、
エンジニアが集まり **発表し、作り、デモする** ワンデイイベントです。
AI エージェントが人間と共にコードを書き、レビューし、テストし、デプロイし、進化させる —
その最前線を探求します。

* **場所:** 未定（神奈川県湘南エリア）
* **日程:** 未定
* **参加費:** 未定（手頃でアクセスしやすい価格を目標）
* **言語:** 日本語・英語（完全バイリンガル）

### フォーマット

イベントは 3 つのパートで構成されます。

1. **プレゼンテーション** — エージェンティック・コーディングに関するディープダイブ・トーク。
   自律的エージェント、ヒューマン・イン・ザ・ループのワークフロー、AI アシスト・アーキテクチャ、
   コード生成のためのプロンプトエンジニアリングなど。
2. **ハッカソン** — 実際に手を動かして作る時間。プロジェクトのアイデアを持ち込むか、
   会場でチームに参加して、エージェンティックなツールと技術を使ってリアルなものを作りましょう。
3. **ハッカソンデモ** — チームが作ったものを発表。ライブデモ、フィードバック、
   そして人間とエージェントが共にコーディングする可能性を讃えましょう。

### 私たちが大切にすること

* **ビルダー、クリエイター、実行する人。** このイベントは、ものを作る人のためのイベントです。
* **技術的な深さと誠実さ。** マーケティングピッチ、投資家向けプレゼン、リクルーティングトークはありません。
* **コミュニティ・ファースト。** スポンサーは適切な形で歓迎しますが、これは企業イベントではありません。
  コミュニティのものです。
* **バイリンガル・バイ・デザイン。** プレゼンテーションや資料は日本語と英語の両方で。
  お好きな言語でご参加ください。

## 湘南エリア

ShoCode を特別にするのは、神奈川県湘南エリアという立地です。
相模湾越しの富士山の眺め、潮風、海岸のリラックスした雰囲気 —
スクリーンの向こうにあるインスピレーションが見つかります。
イベントの前後には、地元のビーチ、トレイル、カフェ、そして湘南ならではの雰囲気をお楽しみください。

## 参加方法

このリポジトリに**プロフィールページ**を提出して参加登録します。
2 つの方法がありますので、お好きなものを選んでください。

### 方法 1: Pull Request（クラシック ShoCode スタイル）

1. **[Creative Commons LICENSE](LICENSE)** をお読みください。
2. **このリポジトリをフォーク**してください。
3. `profiles/` の下に GitHub ユーザー名のフォルダを作成します
   （例: `profiles/your_username`）。
4. Markdown ファイルとしてプロフィールページを追加します。
   [profiles/template.md](profiles/template.md)（英語）または
   [profiles/template_jp.md](profiles/template_jp.md)（日本語）をテンプレートとしてご利用ください。
5. フォークしたリポジトリから本リポジトリへ **Pull Request を送信**してください。

### 方法 2: Copilot に頼む（エージェンティックパス 🤖）

登録用 Issue にあなたの情報をコメントし、**@copilot** に PR の作成を依頼してください。
Copilot があなたのプロフィールページを生成し、PR を代理で提出します。
これは「登録そのものがエージェンティック・コーディング」— AI エージェントと人間が共にコードを書く
カンファレンスにふさわしい登録方法です。

---

### プロフィールページの要件

プロフィールページには以下を記載してください。

* **あなたについて** — 氏名、GitHub ユーザー名、公開の連絡方法を少なくとも 1 つ
  （Twitter/X、LinkedIn、Bluesky、LINE など）
* **何を持ち込むか** — プレゼン、ビルド、またはデモしたい内容の簡単な説明。
  エージェンティック・コーディング、AI アシスト開発、または関連する技術トピックにフォーカスしてください。
* **コードやプロジェクトのリンク** — リポジトリ、Gist、ブログ、過去の成果物を共有してください
* **希望する役割:**
  * プレゼンター — プレゼンテーション枠でトークを行う
  * ハッカー — ハッカソンに参加する（作ったものをデモ）
  * 両方 — プレゼンもハックも
  * 参加者 — 来場し、学び、つながる

PR がレビューされてマージされたら、参加確定です。ShoCode へようこそ！
