- **2026-03-31:** Revised schedule to v2 per Drew's feedback — pushed start from 09:30 to 10:30 so Tokyo attendees avoid rush-hour trains (7:30–9:00am). Compressed morning blocks (opening, keynote, talks, breaks) to absorb the 1hr shift while keeping the 2.5hr hackathon build time fully protected. End time actually improved from 18:30 to 18:00. Key learning: **always factor in commute realities for the primary audience when setting event times** — a 09:30 start in Shonan is really a 08:30 departure from Tokyo, which hits peak rush hour. Tighter presentation slots (15 min vs 20 min standard talks) are a reasonable trade-off for a builder audience that values density.
