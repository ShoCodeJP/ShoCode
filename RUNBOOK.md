## Pre-Event Checklist / 事前準備チェックリスト

### 8+ Weeks Before / 8週間以上前

- [ ] Venue confirmed and contract signed / 会場確定・契約締結
- [ ] Date and time locked / 日程確定
- [ ] Budget finalized / 予算確定
- [ ] Registration page live / 参加登録ページ公開
- [ ] Speaker CFP opened or invitations sent / スピーカー募集・招待送付

### 4–8 Weeks Before / 4〜8週間前

- [ ] AV requirements documented (projectors, screens, mics, adapters) / AV機器要件リスト作成
- [ ] Catering vendor confirmed; menu and dietary needs collected / ケータリング業者確定・メニュー・食事制限の収集
- [ ] Signage designed (wayfinding, schedule board, sponsor logos) / サイネージデザイン（案内表示・スケジュール・スポンサーロゴ）
- [ ] Badge/lanyard design finalized / 名札・ストラップデザイン確定
- [ ] WiFi capacity confirmed with venue (load-test if possible) / WiFi容量を会場と確認（負荷テスト推奨）
- [ ] Power strip and extension cable inventory / 電源タップ・延長コード在庫確認
- [ ] Volunteer team recruited and briefed / ボランティアチーム募集・説明

### 1–2 Weeks Before / 1〜2週間前

- [ ] Final attendee list exported and printed / 最終参加者リスト出力・印刷
- [ ] Badges printed and sorted alphabetically / 名札印刷・アルファベット順に整理
- [ ] Speaker run-of-show shared with all presenters / スピーカーへタイムテーブル共有
- [ ] Speaker slide decks collected (or confirmed on USB) / スピーカースライド収集（またはUSB確認）
- [ ] Dry-run of AV setup at venue (if possible) / AV機器事前テスト（可能なら会場で）
- [ ] Catering final headcount submitted / ケータリング最終人数提出
- [ ] Emergency contact list compiled / 緊急連絡先リスト作成
- [ ] Floor plan and room layout finalized / フロアプラン・レイアウト確定
- [ ] Green room / speaker prep area arranged / スピーカー控え室の手配

### Day Before / 前日

- [ ] Load-in of materials: signage, badges, cables, power strips / 資材搬入：サイネージ・名札・ケーブル・電源タップ
- [ ] Confirm vendor arrival times for day-of / 業者の当日到着時間確認
- [ ] Charge all devices (laptops, clickers, backup batteries) / 全デバイス充電（PC・クリッカー・バッテリー）
- [ ] Print day-of runsheet for ops team / 当日進行表を運営チーム用に印刷

## Setup Checklist / 設営チェックリスト

### AV & Tech / AV・技術

- [ ] Projector(s) powered on and tested / プロジェクター起動・テスト
- [ ] Screen(s) positioned and visible from all seats / スクリーン配置・全席から視認確認
- [ ] Microphones tested (handheld + lapel if available) / マイクテスト（ハンドヘルド・ピンマイク）
- [ ] Speaker laptop adapters ready (HDMI, USB-C, DisplayPort) / スピーカー用アダプター準備
- [ ] Livestream / recording equipment set up (if applicable) / 配信・録画機器セットアップ
- [ ] Backup laptop loaded with all slide decks / バックアップPC にスライド全件格納
- [ ] Clicker devices tested with fresh batteries / クリッカーのバッテリー確認・テスト

### Power / 電源

- [ ] Power strips placed at all hack tables / ハックテーブル全台に電源タップ配置
- [ ] Extension cables routed safely (taped down) / 延長コードを安全に配線（テープ固定）
- [ ] Verify circuit capacity — avoid overloading / 回路容量確認・過負荷防止

### Signage / サイネージ

- [ ] Welcome banner at entrance / 入口にウェルカムバナー
- [ ] Wayfinding signs (rooms, restrooms, WiFi info) / 案内板（部屋・トイレ・WiFi情報）
- [ ] Schedule board in main area / メイン会場にスケジュールボード
- [ ] Code of Conduct posted visibly / 行動規範を見える場所に掲示

### Check-in Desk / 受付

- [ ] Badge boxes sorted and accessible / 名札ボックスを整理・配置
- [ ] Attendee list (digital + printed backup) / 参加者リスト（デジタル＋印刷バックアップ）
- [ ] Swag / materials for distribution / グッズ・配布資料
- [ ] Volunteers briefed on check-in process / ボランティアに受付フロー説明

### Hack Space / ハックスペース

- [ ] Tables and chairs arranged / テーブル・椅子配置
- [ ] Power at every table / 全テーブルに電源
- [ ] WiFi network name and password posted / WiFiネットワーク名・パスワード掲示
- [ ] Whiteboards or flip charts available / ホワイトボードまたはフリップチャート準備

### Green Room / Speaker Prep / スピーカー控え室

- [ ] Quiet space designated for speakers / スピーカー用の静かなスペース確保
- [ ] Water, coffee, light snacks / 水・コーヒー・軽食
- [ ] Power outlets and WiFi / 電源・WiFi
- [ ] Mirror (optional) / 鏡（任意）

## Teardown Checklist / 撤収チェックリスト

- [ ] Pack all AV equipment; check inventory against load-in list / AV機器梱包・搬入リストと照合
- [ ] Remove all signage / サイネージ撤去
- [ ] Collect leftover badges and materials / 残りの名札・資材回収
- [ ] Return venue furniture to original layout / 会場の家具を元の配置に戻す
- [ ] Clean all surfaces and dispose of trash / テーブル清掃・ゴミ処理
- [ ] Check for lost-and-found items / 忘れ物確認
- [ ] Collect feedback (paper forms or QR-code survey link) / フィードバック収集（紙またはQRコードアンケート）
- [ ] Take group photo (before attendees leave!) / 集合写真撮影（参加者が帰る前に！）
- [ ] Final walkthrough with venue contact / 会場担当者と最終確認
- [ ] Return keys / access cards / 鍵・アクセスカード返却

### Contingency Plans / 対応計画

| Scenario / シナリオ | Response / 対応 |
|---|---|
| **Projector failure / プロジェクター故障** | Switch to backup laptop screen-share; venue may have a spare. Always bring a backup projector or confirm venue has one. / バックアップPCの画面共有に切替。会場の予備確認。予備プロジェクター持参推奨。 |
| **WiFi outage / WiFi障害** | Activate mobile hotspot(s); ops team carries at least 2 pocket WiFi units. / モバイルホットスポット起動。ポケットWiFiを最低2台携行。 |
| **Power outage / 停電** | Contact venue immediately; move to well-lit area. Have flashlights. Battery packs power critical laptops. / 会場に即連絡。明るい場所へ移動。懐中電灯準備。バッテリーパックで重要PCに給電。 |
| **Speaker no-show / スピーカー欠席** | Extend hackathon time or pull lightning talk from waitlist. Always have 1–2 backup speakers. / ハッカソン延長またはウェイトリストからLTを追加。バックアップスピーカーを1-2名確保。 |
| **Catering delay / ケータリング遅延** | Extend morning session by 15 min; have emergency snacks on hand. / 午前セッションを15分延長。非常用スナック準備。 |
| **Medical emergency / 医療緊急事態** | Call 119. Identify first-aid trained volunteers in advance. Keep first-aid kit at check-in desk. / 119番通報。事前に応急手当可能なボランティアを特定。受付に救急箱設置。 |
| **Severe weather / 悪天候** | Monitor forecast 48h before. Communicate early if event is affected. Indoor backup plan only. / 48時間前から天気予報監視。影響がある場合は早期連絡。屋内バックアッププランのみ。 |

## Post-Event / イベント後タスク

### Within 48 Hours / 48時間以内

- [ ] Send thank-you message to all attendees / 参加者全員にお礼メッセージ送信
- [ ] Send thank-you to speakers, sponsors, volunteers / スピーカー・スポンサー・ボランティアにお礼
- [ ] Share feedback survey (if not collected on-site) / フィードバックアンケート送信（現地未収集の場合）
- [ ] Upload group photo to repository / event site / 集合写真をリポジトリ・イベントサイトにアップロード

### Within 1 Week / 1週間以内

- [ ] Reconcile vendor invoices against budget / 業者請求書と予算の照合
- [ ] Process vendor payments / 業者への支払い処理
- [ ] Upload presentation recordings (if recorded) / プレゼン録画のアップロード（録画した場合）
- [ ] Publish event recap blog post / write-up / イベントレポート記事の公開

### Within 2 Weeks / 2週間以内

- [ ] Hold ops team retrospective / 運営チーム振り返りミーティング
- [ ] Document lessons learned in repository / リポジトリに学びを記録
- [ ] Update this runbook with improvements / 改善点でこのランブックを更新
- [ ] If rebooking: secure next event date with venue / 次回開催日を会場と仮押さえ
